Margin and padding

Корисна інформація

ВАЖЛИВО! Видалення фантомних відступів між елементами.

Для батьківського елементу вказуємо font-size: 0;-це прибере фантомний відступ між елементами яким вказано display: inline-block;

Для дочірніх елементів потрібно вказати відповідний розмір шрифту, приклад font-size: 16px;

inlin(block)-size.

inline-size - рядковий розмір;

max-inline-size - максимальний рядковий розмір;

block-size - мінімальний блочний розмір

Властивість vertical-align

ivertical-align— керує вирівнюванням блочно-рядкових та рядкових елементів по вертикалі (осі - Y) -! Не успадковується.

  • top— вирівнює по верхньому краю
  • middle— вирівнює по центру
  • bottom— вирівнює по нижньому краю

Cтилі для обгортки (списку) в якій картки

стилі для обгортки (списку) в якій картки / .cards-items { margin: 0 -12px; /відємні відступи margin з боків / font-size: 0; /прибирання фантомних відступів/ }

стилі для карток / .cards-item { display: inline-block; padding: 12px 12px; /встановлення padding для компенсації від'ємного margin і створення відступу між картками/ font-size: 16px; /встановлення розміру шрифту тому що ми для .cards-items встановили розмір шрифту 0*/ };
