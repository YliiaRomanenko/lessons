Стилі шрифтів

Стилі шрифтів-font-style+font-weight (насиченність шрифту)+text-align(вирівнювання тексту)

  • normal (за замовчуванням) - звичайний (не змінений, стандартний) стиль + Жирність font-weight: normal

  • italic - курсив + Жирність font-weight: 700

  • oblique - нахилення літер+ Жирність-font-weight: bold

text-decoration - додає тексту певне оформлення

  • text-decoration-line - встановлює тип декоруу

    • none - (за замовченням) скасовує встановлений раніше параметр.
    • overline - лінія над текстом;
    • underline - лінія під текстом;
    • line-through - закреслює текст;

  • text-decoration-style - змінює стиль декору

  • text-decoration-color - змінює колір декору
